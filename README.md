# ezpbars frontend

frontend for ezpbars using html + css + [vanillaplusjs](https://vanillaplusjs.com/)

## Getting Started

```sh
python -m venv venv
. venv/bin/activate
python -m pip install -U pip
pip install vanillaplusjs
vanillaplusjs init
vanillaplusjs dev --watch
```
